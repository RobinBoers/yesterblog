# Yesterblog

I created this tool to make it easy to host and manage a blog on [Yesterweb.org](https://yesterweb.org/)

It uses PHP 7.4.

# Features
- "Blogroll" homepage
- Login authentication
- Dashboard overview
- Approval page
- Blog comments
- Submission page
- List page
